# 바닐라JS로 크롬 앱 만들기

## “모멘텀” 클론 코딩

---

**갖고 있는 기능** 

- 로그인( 사용자가 입력한 이름 저장)
- 시계
- 내 위치
- 내 위치의 날씨
- 무작위로 바뀌는 배경사진
- 무작위로 바뀌는 명언
- 메인 기능(To Do List)
    - 할 일 입력 후, 입력한 내용 투두 리스트에 추가
    - 투두 리스트에서 체크한 항목 지우기
    - 투두 리스트에 할 일 추가하기
    - 투두 리스트에서 할 일 삭제하기

## 필요한 언어

---

- HTML
- CSS
- JAVASCRIPT
